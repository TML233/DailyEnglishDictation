# Daily English Dictation
这是我做每日英语听写练习的记录。

## 文件
每天的内容保存在以天数命名的 Markdown 文件中，文件内容包含：
1. 初次听写时写完句子所用的次数
2. 初次听写的结果
3. 订正后的结果
4. 订正时的注释

### 标记
#### 初次听写
* ~删除线~，拼错的内容，正确拼写追加在后方。
* ___（下划线），已知未听出的内容。

#### 订正后的初次听写
* **粗体**，错误的内容。
* **~粗体带删除线~**，错误且多余的内容
* **___**（粗体下划线），初次听写时未知的未听出内容。

#### 订正结果
* *foo(1)*，注释标注。

订正后，初次听写中错误的内容以粗体表示，缺少的内容以下划线表示。
订正结果中的注释标记以斜体表示。
